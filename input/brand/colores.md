# Paleta de Colores SINODE

## Primarios
- Azul profundo — `#1E3A8A` (ejemplo Tailwind Blue-900)
- Blanco puro — `#FFFFFF`

## Secundarios
- Verde esperanza — `#10B981` (ejemplo Tailwind Green-500)
- Gris neutro — `#6B7280` (ejemplo Tailwind Gray-500)

## Uso
- Azul profundo: títulos, headers principales.
- Blanco: fondo.
- Verde: acentos (botones, links).
- Gris: texto secundario.
